## Global
- setTimeout, setInterval, clearTimeout, clearInterval
- parseInt, parseFloat, isNaN, isFinite
- encodeURI, decodeURI, eval

## Object
- Object.keys, values, entries
- Object.assign, freeze, seal, create

- map, filter, reduce, flat, flatMap
- forEach, find, findIndex
- some, every, includes
- sort, reverse, slice, splice, concat, push, pop, shift, unshift

## String

- includes, startsWith, endsWith
- replace, replaceAll
- toUpperCase, toLowerCase
- trim, trimStart, trimEnd
- padStart, padEnd
- slice, substring

## Number & Math
- Number.isInteger, Number.isNaN
- Math.random, floor, ceil, round, max, min, abs, pow, sqrt

## Date
- new Date(), getTime, toISOString, getFullYear, getMonth, getDate

## Promise / Async
- Promise.resolve, reject, all, allSettled, race, any

## Map / Set
- Map: set, get, has, delete
- Set: add, has, delete

## JSON
- JSON.parse, JSON.stringify

## RegExp
- test, exec, match, replace

## Error
- throw new Error(), try/catch/finally

## TypeScript Types (compile-time)
- Partial, Required, Readonly
- Pick, Omit, Record
- ReturnType, Parameters, Awaited
